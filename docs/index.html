무엇을 하나

"삼성전자" 헤드라인 20건을 열면 대부분 시황 중계·보도자료 재탕·종목을 스쳐 가는 거시 기사입니다. StoryQuant은 그중 펀더멘털을 바꾸는 진짜 사건만 골라, 호재/악재 집계와 AI 판단 근거까지 함께 보여줍니다.

🔎 노이즈 필터

gpt-4o-mini가 헤드라인을 event / reactive / speculative / off_topic 으로 분류 — 잡음을 걸러냅니다.

🧾 사후 설명

"왜 움직였나"를 뉴스로 맥락화. 코스피·섹터 대비 상대수익률로 종목 고유 움직임을 분리.

💬 판단 근거 공개

호재/악재로 분류한 이유와 전문가 관점 한줄평을 투명하게 — 블랙박스가 아닙니다.

🔔 새 소식 알림

저장한 종목에 실제 사건·큰 변동이 생기면 알려줍니다 (알림 피로 방지).

정직하게 검증했습니다

  • ✗합성 뉴스 점수 — 예측력 미확인
  • ✗이벤트 인과 — placebo로 반증
