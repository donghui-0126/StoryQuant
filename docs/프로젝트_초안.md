# 📌 프로젝트 초안: QuantFlow

**뉴스 기반 멀티에셋 Hot Topic & Price Move Attribution Dashboard**

## 1. 프로젝트 개요

실시간 뉴스 데이터를 수집하여 **1시간 단위 Hot Topic을 추출**하고,
크립토 / 국장 / 미장 주요 자산에서 **유의미한 가격 변동 이벤트 발생 시 그 원인을 자동으로 설명하는 대시보드**를 구축한다.

단순 뉴스 요약이 아니라,

> **“시장 가격이 왜 움직였는가?”**
> 를 정량적으로 설명하고 투자 의사결정에 활용 가능한 형태로 제공하는 것이 목표다.

---

## 2. 문제 정의

투자 의사결정에서 중요한 것은 단순 뉴스 소비가 아니라 아래 2가지다.

### (1) 현재 시장의 핵심 Hot Topic

* 지난 1시간 동안 가장 많이 언급된 테마
* 자산군별 주요 이슈
* topic persistence (지속성)
* macro / sector / idiosyncratic 분리

### (2) 특정 가격 변동의 원인 설명

예:

* BTC 급등 → ETF 관련 뉴스
* 삼성전자 급락 → 실적 가이던스 하향
* NVDA 급등 → AI capex 관련 발언

즉,

> **뉴스 이벤트 ↔ 가격 움직임 attribution**
> 을 자동화한다.

---

## 3. 핵심 기능

## A. Hot Topic Dashboard

1시간 단위로 뉴스 데이터를 집계하여:

* 실시간 topic ranking
* 키워드 클러스터링
* 시장별 topic heatmap
* topic momentum score
* topic novelty score

제공

---

## B. Price Move Attribution

특정 주요 자산에서 가격 변화 이벤트 발생 시:

### 이벤트 트리거 예시

* 1h return > threshold
* abnormal volume
* OI spike
* realized volatility shock

이 발생하면 해당 시간대 뉴스 수집 후

* 관련 뉴스 후보 검색
* 룰 기반 매핑
* LLM 기반 요약
* 주요 원인 confidence score

를 제공

## 5. PoC 범위 (이번 주 일요일)

이번 PoC에서는 **핵심 가설 검증용 MVP**만 구현한다.

### 포함 범위

### 1) 뉴스 크롤링

* crypto: 주요 crypto media / X headline
* 국장: 네이버 금융 / 주요 경제지
* 미장: CNBC / Reuters / earnings headlines

### 2) Hot Topic 추출

* 1시간 단위 키워드 빈도
* TF-IDF / embedding clustering
* 상위 토픽 5개 랭킹

### 3) 가격 이벤트 감지

* 주요 자산 5~10개
* 1시간 수익률 threshold 기반
* 급등락 이벤트 탐지

### 4) 뉴스 원인 매핑

1차:

* keyword overlap
* ticker mention
* sector mapping
* 발표 시간 proximity

2차:

* LLM 요약 실험

## 6. AI 활용 포인트

AI는 아래 부분에서 가장 효율적이다.

### Hot Topic 추출

* 중복 뉴스 제거
* 유사 headline clustering
* 의미 기반 topic labeling

예:

> → **Fed Hawkishness**

> 주요 원인: ETF inflow acceleration 관련 뉴스
> 보조 원인: short squeeze 가능성

## 7. 성공 Metric

투자 목적 프로젝트라 metric이 명확하다.

### 시스템 Metric

* Hot topic precision
* event-news matching accuracy
* LLM attribution relevance
* dashboard latency (<5min)

### 투자 활용 Metric

* topic persistence vs future return
* event attribution 이후 continuation probability
